GitHub Repository URL: https://github.com/Itsjustmeblank/Asset_Placement.git

Description:
The Asset Placement Tool is a Maya utility created with Python and PySide2 that helps artists quickly place objects throughout a scene using procedural placement for personal code or through selection in a scene. The tool supports Random, Ring, Line, Grid, and Box layouts, preview generation, collision avoidance, object instancing, automatic grouping, and surface-based placement on polygon meshes.
